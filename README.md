<h1><span style='font-size:12.0pt;line-height:115%;font-family:"Calibri","sans-serif";
color:black'>Introduction</span></h1>

<p class=MsoNormal style='margin-bottom:0in;margin-bottom:.0001pt;line-height:
normal;text-autospace:none'><span style='font-size:12.0pt;font-family:"Times New Roman","serif"'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>This PollMap Application allows you to capture and map public
opinion on any issue or topic you configure. It encourages participation
through a user experience that focuses on simple things: reading the question, choosing
among the options presented, and then viewing results immediately on the map. This
gives the visitor instant feedback and an option to explore the map to see how
their vote fits within local, regional and even national contexts.</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>This map is known simply as PollMap, or as the Esri FanMap™ when
applied to sports topics. You can see an example of this template at work in
the Esri </span><a href="http://fanmap.esri.com/collegehoops"><span
style='font-family:"Calibri","sans-serif"'>FanMap™: Bracketography Edition</span></a><span
style='font-family:"Calibri","sans-serif";color:black'>.</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'><img border=0 width=882 height=440 id="Picture 4"
src="Getting_Started_for_PollMap_template_files/image002.jpg"></span></p>

<p class=MsoPlainText><i><span style='font-family:"Calibri","sans-serif";
color:black'>Figure 1: Sample PollMap about water.</span></i></p>

<h1><span style='font-size:12.0pt;line-height:115%;font-family:"Calibri","sans-serif";
color:black'>What is included? </span></h1>

<p class=MsoNormal style='margin-bottom:0in;margin-bottom:.0001pt;line-height:
normal;text-autospace:none'><span style='font-size:12.0pt;font-family:"Times New Roman","serif"'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>This is version 1.0 of the PollMap application template. It
supports a basic type of “A vs. B” question and all documentation is written
with that in mind. </span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>This JavaScript application is built using the ArcGIS API for
JavaScript. It contains an interactive map intended for use in a standard web
browser. It does not require browser plug-ins or installs. This application
accesses several map services and geoprocessing services.</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>The template includes sample data to record votes by postal code
in the U.S. and Canada, and map documents (MXDs) to edit their symbology on the
map. Note: it is possible to substitute other point data to gather the votes,
e.g. see the Esri </span><a href="http://pollmap.esri.com/earthday"><span
style='font-family:"Calibri","sans-serif"'>Earth Day Poll Map</span></a><span
style='font-family:"Calibri","sans-serif";color:black'> which uses cities of
the world. That level of customization is not described in this document.</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>The template includes two geoprocessing scripts: one to record
incoming votes and another to summarize those votes by location. </span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>The template includes three map services: one service
(zips_winkel) to provide access to the postal codes to geoprocessing scripts,
and two services (Question01 and Question02) to symbolize the dominant vote
count at each location.</span></p>

<p class=MsoNormal style='margin-bottom:0in;margin-bottom:.0001pt;line-height:
normal;text-autospace:none'><span style='font-size:12.0pt;font-family:"Times New Roman","serif"'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>The template uses a simple multi-scale basemap hosted by Esri for
use by this application. The map is in the Winkel-Tripel projection, centered
on North America. See details about this projection </span><a
href="http://blogs.esri.com/Support/blogs/mappingcenter/archive/2011/02/17/an-alternative-to-web-mercator-winkle-triple.aspx"><span
style='font-family:"Calibri","sans-serif"'>here</span></a><span
style='font-family:"Calibri","sans-serif";color:black'>. If you to deploy this
template in a site which may attract more than 100 visitors an hour, please </span><a
href="mailto:jherries@esri.com"><span style='font-family:"Calibri","sans-serif"'>contact
us</span></a><span style='font-family:"Calibri","sans-serif";color:black'> so
that we can work with you to arrange proper support.</span></p>

<h1><span style='font-size:12.0pt;line-height:115%;font-family:"Calibri","sans-serif";
color:black'>How to configure and use the PollMap Template</span></h1>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>First, you will get the sample PollMap up and running on your
machine. You will configure the PollMap template for your environment, and in
doing so, learn how to publish and serve your own poll maps using ArcGIS Server
and your organization’s data. Then, the second set of steps outlines how to
customize the PollMap to contain your questions, choices and symbology for the
maps and charts.</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>&nbsp;</span></p>

<p class=MsoPlainText><span style='font-family:"Calibri","sans-serif";
color:black'>To complete these steps, you will need experience with ArcGIS
Server and Microsoft’s Internet Information Server (IIS). If you are new to
JavaScript applications, this template will demonstrate a pattern you can use
to deploy your own JavaScript application and publish web maps using ArcGIS
Server.</span></p>
