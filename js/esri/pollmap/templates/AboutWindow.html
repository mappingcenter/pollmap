About the Earth Day PollMap

Help us observe Earth Day 2011 by letting us know what you think are the most important environmental challenges facing humankind. Imagine you had $100 to spend to help protect the Earth. How would you invest your money? Current statistics and charts are updated as you look around the map. The map is something of an experiment: Will people from different countries have differing priorities? See if you can discern any regional patterns.

Basemap

This basemap provides a neutral backdrop for thematic maps like the PollMap. The map allows the key information to come to the foreground while the rest of the map provides simple geographic context. Administrative boundaries and major water features help create that context along with city boundaries and labels. The projection used in this version is Web Mercator Auxiliary Sphere.

Layers

Graduated circles are used to represent the current vote totals and trends by city. Color and size are assigned based on plurality vote in each city.

Application

This application is powered by ArcGIS Server. It uses the ArcGIS JavaScript API as the client. The FanMap template was the starting point for this version, with changes made to accommodate a new voting type (allocating dollars across several options) and new basemap.
